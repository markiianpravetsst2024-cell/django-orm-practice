# Cinema Project (Django ORM + PostgreSQL)



# Опис моделей 
**Python**: 3.14.0 
**Django**: 6.0.3
**Інструменти**: `python-dotenv` (для безпеки), `psycopg2-binary`




## Як запустити проект
1. Склонуйте репозиторій.
2. Створіть файл `.env` у корені та додайте ваші дані для PostgreSQL.
3. Встановіть залежності: `pip install django python-dotenv psycopg2-binary`.
4. Виконайте міграції: `python manage.py migrate`.
5. Запустіть запити: `python queries.py`.
